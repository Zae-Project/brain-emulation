Documentation 12 · Glossary & Further Resources

Use this glossary as a quick reference while navigating the simulator and reading the documentation. The resource section collects foundational papers and tools that informed the project.

Glossary

Atlas Template
A JSON configuration describing clusters, neuron groups, and interconnections. See data/brain_region_maps/.
Cluster
Logical grouping of neurons with shared location and connectivity rules.
Excitatory/Inhibitory Ratio (E/I)
Proportion of excitatory to inhibitory neurons. Human cortex ~80/20.
Glyph
Geometric shape representing a neuron type in the canvas and inspector.
Manifest
File that indexes available templates (data/brain_region_maps/manifest.json).
Neuron Group
Entry within a cluster specifying the neuron preset slug and count (e.g., pyramidal × 120).
Preset
Entry in the dropdown that references a template. Includes builtin and imported configurations.
Pulse Decay
Visual parameter controlling how long neuron glow persists after firing.
SNN_CONFIG_IO
Utility module responsible for validating, normalizing, and serializing templates.
SNN_REGISTRY
Runtime registry that tracks neuron presets, templates, and exposes helper methods to the UI.
Voltage Trace
Rolling plot of membrane potential for the selected neuron, updated every frame.
